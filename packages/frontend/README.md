# @montemove/frontend

Frontend приложение на Next.js 15.

## Разработка

```bash
# Из корня монорепо
pnpm dev:frontend

# Или напрямую
cd packages/frontend
pnpm dev
```

Приложение на [http://localhost:3001](http://localhost:3001)

## Команды

```bash
pnpm dev          # Dev сервер (порт 3001)
pnpm build        # Сборка
pnpm start        # Продакшн сервер
pnpm lint         # Линтинг
```

## Стек

Next.js 15, React 19, TailwindCSS 4, Radix UI, shadcn/ui, Zustand, React Query
